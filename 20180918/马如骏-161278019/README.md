## 实现了三个类
- BalashBrother
    
    葫芦娃枚举类型，包含名字，颜色，序号，位置（对位置属性有点疑惑）
    - TellName() 葫芦娃报自己的名字
    - TellColor() 葫芦娃报自己的颜色
    - Move() 葫芦娃移动到新的位置
- BalashBrothers 
    
    葫芦娃队列，可以给葫芦娃排好随机的顺序，并让他们站下
- Sorter 

    给葫芦娃排序，有冒泡排序和二分排序两种

## 存在的疑问
枚举类型的地址是固定不变的，但如果我想要改变枚举类型中的某些属性（比如A队列中蓝娃的位置），就导致其他的枚举葫芦娃的属性（B队列中蓝娃的位置）也改变了。所以枚举类型中是否应该存在可改变的属性。

给枚举类型定义的位置属性，和葫芦娃队列中的数组下标含义相同，感觉有些冗余，但又觉得不可缺少。
